A Linguagem HTML

HTML (abreviação de expressão inglesa HyperText Markup Language, que significa Linguagem de Marcação de Hipertesto) é uma linguagem de marcação utilizada na marcação de páginas na Web. Documentos HTML podem ser interpretados por navegadores.

Alguns dos recursos da linguagem HTML vistos até este momento:

Recursos Tags assossiadas Descrição
Links <a> Criar links entre páginas
Listas <ul> e <li> Criar lista de itens ordenados
<ol> e <li> Criar listas de itens não ordenados
Imagens <img> Inserir imagens
